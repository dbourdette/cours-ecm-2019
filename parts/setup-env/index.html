Mise en place de l'environnement

Afin de pouvoir développer, nous avons besoin de quelques outils :

  • Environnement java (jdk)
  • Outil de build (gradle)
  • Spring boot CLI (client line interface)

SDKMAN!

Le plus simple est passer par SDKMAN!

Cet outils permet de gérer les différentes versions d'autres logiciels de l'environnement de développement.

➤ Installez sdkman en suivant le guide d'installation

Windows

Sur Windows, l'installation est plus douloureuse. Voici ce que je propose :

  • installer Git Bash for Windows
  • installer la commande zip via https://sourceforge.net/projects/gnuwin32/files/ (lire add-zip-into-git-bash-on-windows)

Sur ma machine, les fichiers zip.exe et bzip2.lib vont dans C:\Program Files\Git\usr\bin
