The bookshelf

Création d'un app bookshelf avec dépendance mongodb

Spécifications

Gestion de livres et de commentaires

Modèle de données

Livre : titre, auteur, isbn

Commentaires : date, isbn, texte

Remplir la base

Code pour remplir la base

Chemins

  • /books : liste par ordre alphabétique de titre.
  • /books/[isbn] : un livre
  • /books/[isbn]/comments : tous les commentaires sur un livre par ordre chronologique
  • /books/[isbn]/comments/[id] : un commentaire sur un livre
  • /comments : liste par ordre chronologique.
  • /comments/[id] : un commentaire

Ajout de livre et de commentaires

Mettez en place le modèle de données, les repositories et les controllers

Fonctionnalités

  • /books : liste paginée par 50 (paramètres page=xxx)
  • /books : recherche par titre
  • /books/[isbn]/comments/[id] : que faire si l'id ne correspond pas a l'isbn ?
  • /comments : liste paginée par 50 (paramètres page=xxx)
  • Pouvoir ajouter un commentaire (POST)
  • Pouvoir ajouter un livre (POST)
  • Pouvoir modifier un livre (PUT)
  • Pouvoir supprimer un livre (DELETE). Il faut supprimer les commentaires qui en dépendent.
