Ce sont des services disponibles sur le réseau

Historiquement, c'est SOAP (Simple Object Access Protocol) qui a été le premier support pour la démocratisation des web services

SOAP repose exclusivement sur le XML et peut fonctionnre par dessus plusieurs protocoles d'échange: HTTP, SMTP...

En pratique, SOAP est lourd et complexe.

Il est remplacé par des services plus légers

  • uniquement HTTP
  • le json est le format privilégié

Une bonne connaissance du HTTP est donc essentielle

Il existe 2 types de services

  • RPC (comme SOAP)
  • REST

4 opérations du REST

  • POST
  • PUT
  • GET
  • DELETE
