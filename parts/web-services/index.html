Web services

Ce sont des APIs disponibles en HTTP

Il existe 2 types de web services

  • RPC : remote procedure call
  • REST : representational state transfer

Le RPC c'est l'appel d'une méthode à distance

                        String sayHello(String name)

Comme si elle était dans le même programme

Le REST permet d'interagir des ressources

                        GET https://mydomain.com/customers/raoul/accounts

Une url correspond à une ressource

Il y à 4 opérations pour faire du REST

  • GET : récupérer la ressource
  • POST : ajouter une ressource
  • PUT : modifier une ressource
  • DELETE : supprimer une ressource

C'est aussi le clasique CRUD des bases de données

Le format privilégié pour les web services est

{"employees":[
